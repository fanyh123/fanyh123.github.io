MacOS 安装配置PyV8

· 2019-09-04 · # python

pyv8是一个纯粹的非基于浏览器的JavaScript运行时环境.
Python3 安装不要使用pip，因为官方只支持 Python2，需要在这里下载对应系统的二进制文件：github.com/emmetio/pyv…
然后解压后将 PyV8.py 与 _PyV8.so (如so不是这个名字需要改成这样) 两文件复制到 Python 的 site-packages 目录下，如 /usr/local/lib/python3.6/site-packages 。

安装Boost依赖:

brew install boost 

brew install libboost-all-dev boost-python3 boost-build boost-mpi

安装V8环境:

brew install v8

安装pyv8:

pip install -e git://github.com/brokenseal/PyV8-OS-X#egg=pyv8

代码示例:

# -*- coding: utf-8 -*-

from pyv8 import PyV8

ctxt = PyV8.JSContext()
ctxt.enter()

# 函数
func = ctxt.eval("""
    (function(){
        function hello(){
            return "Hello world.";
        }
        return hello();
    })
""")
print func()
# Hello world.

# 变量
ctxt.eval("""
    var_ex1 = 1;
    var_ex2 = 1.0;
    var_ex3 = "test";
    var_ex4 = true;
    """)

vars = ctxt.locals
print vars.var_ex1
# 1

# 传入参数
ctxt.locals.test = 12
print int(ctxt.eval("test"))
# 12
